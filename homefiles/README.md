# .bash_aliases
- Adds 3 useful aliases:
	- alias cl="clear"
	- alias h="history"
	- alias web='cd /var/www/html'

# setup_aliases.sh
- This script makes a backup of any existing .bash_aliases in the user's home directory, then symbolically links the .bash_aliases file in this repository to your user's home directory
