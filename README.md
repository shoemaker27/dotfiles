# My Dotfiles
- This repo contains the following scripts:
	- homefiles/setup_aliases.sh
		- Updates the system with my .bash_aliases
	- sys-installs/install_aws_cli.sh
		- Silently installs AWS CLI
	- vimInstall.sh
		- Installs Vundle and my Vundle Plugins

- How to Use:
	- Clone repo: `git clone git@github.com:shoemaker27/dotfiles.git`
	- Run setup_aliases.sh: 
		- `cd homefiles`
		- `./setup_aliases.sh`
	- Run install_aws_cli.sh: 
		- `cd sys-installs`
		- `./install_aws_cli.sh`
	- Run vimInstall.sh: `./vimInstall.sh`
	
	- More details about each script can be found in the README.md files in their respective folders
